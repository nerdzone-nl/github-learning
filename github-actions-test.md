## GitHub Actions
GitHub biedt de mogelijkheid om met [GitHub Actions](https://docs.github.com/en/actions) wat automatisering aan een project toe te voegen. Om een beetje inzicht te krijgen in hoe dit werkt is de volgende workflow bedacht:
 * Indien aanwezig, verwijder de oude zipfile, met dezelfde naam als dit bestand
 * Voeg een regel toe aan dit bestand toe met een datum-tijd combinatie met het tijdstip van uitvoering;
 * Zip dit bestand en voeg het opnieuw toe aan de repository
